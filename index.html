How to Play?

Welcome to The Simon Game

Introducing an exicitng memory game - Are you up for the challenge?

  • Turn the game on usng the POWER button.
  • Press START to begin the game.
  • The game will show you a sequence which you must memorise.
  • Repeat the sequence to get to the next level. Each level will be a bit more challenging. If you get the sequence wrong, you can keep trying to get to the next level.
The Strict Button

If you are ready for an even greater challenge turn on the STRICT button.
This means that you cannot get an answer wrong or it is GAME OVER.
The game is won once you have reached 20!
